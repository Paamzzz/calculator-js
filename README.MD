# 🧮 JavaScript Calculator

A simple calculator built with **HTML, CSS and Vanilla JavaScript**.

This project was created as a practice exercise while revisiting JavaScript fundamentals such as DOM manipulation, event handling and application state.

## 🔗 Live Demo

Try it here:  
https://js-calculator-indol.vercel.app

## 🛠️ Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## ✨ Features

- Basic arithmetic operations
- Clear button to reset the calculator
- Sequential operations (like traditional calculators)
